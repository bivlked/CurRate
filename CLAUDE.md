# CurRate - Конвертер валют с курсами ЦБ РФ

## Описание проекта
GUI приложение на Python (Tkinter) для конвертации USD/EUR в рубли по официальным курсам Центрального Банка РФ.

### Основные функции:
- Парсинг курсов валют с сайта cbr.ru
- Графический интерфейс с календарем
- Конвертация на выбранную дату
- Копирование результата в буфер обмена

## Технологический стек
- **Python**: 3.14+
- **GUI**: Tkinter (встроен в Python)
- **Зависимости**:
  - `requests` - HTTP запросы к сайту ЦБ РФ
  - `beautifulsoup4` - парсинг HTML
  - `tkcalendar` - виджет календаря для Tkinter
  - `pyperclip` - работа с буфером обмена
- **Менеджер зависимостей**: Poetry
- **Структура**: src-layout (src/currate/)

## Архитектура
- `src/currate/` - основной код приложения
- `tests/` - тесты
- Используется Poetry для управления зависимостями
- Виртуальное окружение создается автоматически через Poetry

## Важные команды

### Работа с зависимостями:
```bash
poetry add <package>              # Добавить зависимость
poetry add --group dev <package>  # Добавить dev зависимость
poetry install                    # Установить все зависимости
poetry update                     # Обновить зависимости
poetry show                       # Показать установленные пакеты
```

### Запуск:
```bash
poetry run python src/currate/main.py  # Запуск GUI приложения через poetry
poetry shell                           # Активировать venv
python src/currate/main.py             # Запуск в активированном venv
```

**Примечание**: Приложение открывает GUI окно, не закрывайте терминал во время работы.

### Тестирование:
```bash
poetry run pytest              # Запуск тестов
poetry run pytest -v           # Подробный вывод
poetry run pytest --cov        # С покрытием кода
```
