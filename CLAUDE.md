# CurRate - Проект для работы с курсами валют

## Описание проекта
Проект для получения и анализа курсов валют.

## Технологический стек
- **Python**: 3.14+
- **Менеджер зависимостей**: Poetry
- **Структура**: src-layout (src/currate/)

## Архитектура
- `src/currate/` - основной код приложения
- `tests/` - тесты
- Используется Poetry для управления зависимостями
- Виртуальное окружение создается автоматически через Poetry

## Важные команды

### Работа с зависимостями:
```bash
poetry add <package>              # Добавить зависимость
poetry add --group dev <package>  # Добавить dev зависимость
poetry install                    # Установить все зависимости
poetry update                     # Обновить зависимости
poetry show                       # Показать установленные пакеты
```

### Запуск:
```bash
poetry run python src/currate/main.py  # Запуск через poetry
poetry shell                           # Активировать venv
python src/currate/main.py             # Запуск в активированном venv
```

### Тестирование:
```bash
poetry run pytest              # Запуск тестов
poetry run pytest -v           # Подробный вывод
poetry run pytest --cov        # С покрытием кода
```

## Соглашения по коду
- Стиль: PEP 8
- Форматирование: black
- Именование: snake_case для функций и переменных
- Документация: docstrings для всех публичных функций

## Переменные окружения
Используйте `.env` файл для хранения конфиденциальных данных:
- API ключи
- URLs
- Токены

Смотрите `.env.example` для шаблона.
